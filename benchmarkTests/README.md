# JVectorCUDA Benchmark Results

This directory contains benchmark results generated by `./gradlew benchmark`.

## Output Files

- **benchmark-report.md** - Latest human-readable performance report
- **benchmark-results-TIMESTAMP.csv** - CSV data for regression tracking
- **benchmark-results-TIMESTAMP.json** - JSON data for programmatic analysis

## File Formats

### CSV Format

```csv
timestamp,gpu_name,vector_count,query_count,dimensions,cpu_time_ms,gpu_time_ms,speedup
```

### JSON Format

```json
{
  "benchmark_metadata": {
    "timestamp": "2026-01-03T16:45:31",
    "gpu_name": "NVIDIA GeForce GTX 1080...",
    "dimensions": 384,
    "warmup_iterations": 3,
    "measured_iterations": 5
  },
  "results": [...]
}
```

## Usage

### Running Benchmarks

```bash
./gradlew benchmark
```

### Saving Baseline for Comparison

```bash
# Save current results as baseline
cp benchmark-results-*.csv baseline.csv
```

### Comparing Results

After making changes, run benchmarks again and compare:

```bash
diff baseline.csv benchmark-results-*.csv
```

## Notes

- Timestamp format: `YYYYMMDD_HHMMSS`
- GPU name may contain semicolons in CSV (replacing commas)
- Speedup > 1.0 means GPU is faster than CPU
- Keep baseline.csv for regression detection
